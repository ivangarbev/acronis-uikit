# Design Tokens Technical Reports — 2025.10

## Abstract

Design tokens are indivisible pieces of a design system such as colors, spacing, typography scale.

Design tokens were created by the Salesforce design system team, and the name comes from them ([Jon](https://x.com/jonnyl) & [Jina](https://x.com/jina)).

## Modules

The Design Tokens specification is composed of multiple modules:

- [Format](./format/index.md) — file format for exchanging design tokens between tools.
- [Color](./color/index.md) — color values and opacity.
- [Resolver](./resolver/index.md) — working with design tokens in multiple contexts (e.g. light/dark themes).
